Hosting & URL parameters
  • Place your groups file in /groups/ (default: groups/Class1.txt).
  • Place your images in /pics/ (default folder). File base names must match names in the txt.
  • Supported extensions (priority): .jpg → .jpeg → .png.
  • You can override defaults via query params:
    • ?group=groups/MyClass.txt
    • ?pics=pics (or any folder name)
