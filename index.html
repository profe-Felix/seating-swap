How mapping works
  • If a student name equals a slot id (e.g., 'A1'), that student is placed in that slot.
  • Otherwise, students are placed in slots sequentially by order.
  • Click two seats to swap where those two students sit (updates the groups model).
  • Use Edit layout to drag/resize/rotate seats; then Export Layout JSON.
